## 如何使用 checkin_sites

在 `LINUXDO_ACCOUNTS` 环境变量（JSON 数组）中为每个账号添加 `checkin_sites` 字段：

```json
[
  {
    "username": "your_email@example.com",
    "password": "your_password",
    "name": "主账号",
    "checkin_sites": ["kfcapi", "duckcoding", "hotaru"]
  }
]
```

- `checkin_sites` **不设置 / 空列表** → 签到全部可用站点
- `checkin_sites` **非空** → 仅签到列表中的站点
- 站点 ID 大小写不敏感（`KFCapi` 和 `kfcapi` 效果一样）
